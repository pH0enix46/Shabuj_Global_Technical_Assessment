# 🎓 University Finder Module

A highly optimized "Find University" module built with Next.js (App Router) and Tailwind CSS.  
This project is designed to provide students with a fast, SEO-friendly, and user-focused experience when searching for universities abroad.

---

## 🚀 Tech Stack

- Next.js (App Router)
- Tailwind CSS
- Next.js API Routes
- Prisma and Neon for Database
- Dummy Database (Real University Names)

---

## 📊 Core Features

### 1️⃣ University Data

Each university contains:

- University Name
- Country
- Location (City/State)
- Tuition Fee
- Ranking
- Established Year

---

### 2️⃣ Server-Side Filtering

All filtering logic is handled on the backend for better performance and scalability

Available Filters:

- Country
- City/State
- Tuition Fee (Range)
- Ranking (Range)
- Established Year (Range)

---

## ⚡ Performance & Optimization

- Server-side data filtering
- Optimized rendering strategy
- SEO-friendly structure (proper metadata, semantic HTML)
- Responsive design for all devices
- Clean and modular code architecture

---

## 🆚 Bonus Feature (Optional)

### Compare Universities

Users can select two universities and view a side-by-side comparison inside a modal.
